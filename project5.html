Weather App

Project Overview

A user-friendly tool designed to help students accurately calculate their CGPA and GPA based on Anna University's grading system. It features an intuitive interface, allowing users to input grades effortlessly and instantly view results, ensuring precision and ease of use.

Key Features

  • Responsive Design
  • User Engagement
  • Menu

Frontend: HTML, CSS, JavaScript

Future Enhancements

Add a feature that allows students to input expected grades for ongoing semesters to predict their final GPA or CGPA.

Include options to save and track GPA scores for multiple semesters, providing an overview of academic progress over time.

Enable offline functionality to calculate GPA/CGPA without requiring an internet connection.
