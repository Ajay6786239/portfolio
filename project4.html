Weather App

Project Overview

A responsive weather app that displays real-time temperature for any location worldwide. Integrated light and dark mode features for user preference, along with the ability to toggle between Celsius and Fahrenheit. The app offers a clean and interactive user interface, enhancing the user experience.

Key Features

  • Responsive Design
  • User Engagement
  • Menu

Frontend: HTML, CSS, JavaScript, Weather API

Future Enhancements

Automatically detect the user's location and display the local weather.

Enable users to check the weather via voice commands.

Provide limited functionality like previously fetched data when the app is offline.
